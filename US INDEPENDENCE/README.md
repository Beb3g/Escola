# Site - Independência dos Estados Unidos
## Colégio Integrado Cesurg - Marau, RS

### Como Adicionar a Logo do Colégio

1. **Obtenha a imagem da logo** do Colégio Integrado Cesurg
   - Formato recomendado: PNG com fundo transparente
   - Tamanho mínimo: 200x200 pixels
   - Formato máximo: 600x600 pixels

2. **Coloque a imagem na pasta do projeto**
   - Coloque o arquivo na mesma pasta que contém o `index.html`
   - Renomeie o arquivo para `logo.png`

3. **Se quiser usar um nome diferente**
   - Abra o arquivo `index.html`
   - Procure por: `<img src="logo.png"` (linha aproximadamente 340)
   - Substitua `"logo.png"` pelo nome do seu arquivo
   - Exemplo: `<img src="cesurg-logo.png"`

- Abra o arquivo `index.html` no seu navegador
- Clique nos botões "English" / "Português" para alternar o idioma
- A preferência de idioma é salva automaticamente
